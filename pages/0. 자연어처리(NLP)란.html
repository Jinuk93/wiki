💬 자연어 처리(NLP) & 자연어 이해(NLU), 그게 뭔가요?

자연어 처리 (Natural Language Processing, NLP) :
컴퓨터에게 사람의 언어를 다루는 법을 가르치는 전반적인 과정

자연어 이해 (Natural Language Understanding, NLU) :
NLP의 하위 분야로, 조금 더 깊이 들어가는 개념으로서, 컴퓨터가 단순히 텍스트를 처리하는 것을 넘어, 문장의 진짜 의미와 의도를 파악하는 데 중점

"오늘 날씨 어때?" → '날씨 정보를 물어보는구나'라고 핵심 의도를 파악

🗺️ 앞으로 우리가 함께 탐험할 분야들

텍스트 전처리
: 컴퓨터가 텍스트를 잘 이해할 수 있도록 재료를 다듬는 과정 (토큰화, 정제, 정규화 등)

단어 표현(임베딩)
: '사랑', '우정' 같은 단어들을 컴퓨터가 계산할 수 있는 숫자(벡터)로 바꾸는 기술
(Word2Vec, GloVe, FastText 등)

다양한 NLP/NLU 모델
: 문장의 의미를 더 깊게 이해하는 인공지능 모델 (RNN, LSTM, Transformer 등)

응용 분야
: 챗봇, 번역기, 감성 분석, 텍스트 요약 등

파이프라인
정제 → 토큰화 → 벡터화/임베딩 → 모델 → 평가/배포
