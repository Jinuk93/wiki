💬 자연어 처리(NLP) & 자연어 이해(NLU), 자연어 생성(NLG)이란?

1

자연어 처리 (NLP) : 컴퓨터와 언어로 대화하기

자연어 처리(Natural Language Processing, NLP)는 컴퓨터가 인간의 언어(자연어)를 이해하고, 생성하며, 다룰 수 있도록 하는 인공지능의 한 분야입니다. NLP는 컴퓨터에게 언어 능력을 가르치는 모든 과정을 포괄하는 가장 넓은 개념입니다.

언어를 담당하는 '뇌(Brain)' 🧠

NLP는 언어와 관련된 모든 것을 처리하는 인간의 '뇌'와 같습니다. 이 뇌는 크게 두 가지 핵심 기능을 수행합니다. 바로 '듣고 이해하는 능력'과 '생각해서 말하는 능력'이죠.

2

자연어 이해 (NLU) : 듣고 의미를 파악하는 능력 👂

자연어 이해(Natural Language Understanding, NLU)는 NLP의 하위 분야로, 컴퓨터가 텍스트의 진짜 의미와 의도를 파악하는 데 중점을 둡니다. 즉, 입력(Input)을 이해하는 기술입니다.

‘귀'와 '이해하는 뇌'

NLU는 상대방의 말을 귀로 듣고,
그 말의 핵심 의도가 무엇인지 파악하는 뇌의 작용과 같습니다.

주요 Task 예시

  • 의도 파악 (Intent Classification):
    "오늘 날씨 어때?" → '날씨 정보 질문'이라는 핵심 의도를 파악합니다. (챗봇의 핵심)
  • 개체명 인식 (Named Entity Recognition, NER):
    "내일 부산 날씨 알려줘" → '부산'이 '장소'라는 유형의 단어임을 알아챕니다.
  • 감성 분석 (Sentiment Analysis):
    "이 영화 정말 최고야!" → 문장에 담긴 감정이 '긍정'임을 판단합니다.
3

자연어 생성 (NLG) : 생각해서 문장으로 말하는 능력 ✍️

자연어 생성(Natural Language Generation, NLG) 역시 NLP의 하위 분야로, 컴퓨터가 내부적인 데이터나 이해한 내용을 바탕으로 새로운 문장을 만들어내는 데 중점을 둡니다. 즉, 출력(Output)을 생성하는 기술입니다.

'입'과 '말하는 뇌'

NLG는 머릿속의 생각(데이터)을 바탕으로,
문법과 문맥에 맞게 자연스러운 문장을 조립하여 입으로 말하는 작용과 같습니다.

주요 Task 예시

  • 챗봇 답변 생성 :
    NLU가 '날씨 정보 질문' 의도를 파악하면, NLG는 {서울, 25도, 맑음} 데이터를 받아
    "오늘 서울의 날씨는 25도이며 맑습니다." 라는 완전한 문장을 생성합니다.
  • 기계 번역 (Machine Translation) :
    번역 모델의 디코더 부분이 원문의 의미를 바탕으로 새로운 언어의 문장을 생성합니다.
  • 텍스트 요약 (Text Summarization) :
    긴 기사의 핵심 내용을 이해(NLU)한 뒤, 이를 바탕으로 짧은 요약문을 새로 생성(NLG)합니다.

한눈에 정리하기

사람의 대화로 비유하자면,

  • NLU : 친구의 말을 듣고 "아, 배고프다는 뜻이구나!"라고 이해하는 과정
  • NLG : "그럼 우리 밥 먹으러 갈까?"라고 대답할 문장을 만들어 말하는 과정
  • NLP : 이 모든 과정을 포함하는 '대화' 그 자체

NLP (대화 능력) = NLU (이해 능력) + NLG (생성 능력)
